Programación front end

La programación front-end es la parte de un programa, sitio web o dispositivo a la que un usuario puede acceder directamente. Es la cara visible del sitio, donde se muestra el diseño, los colores, los botones y todo lo que puedes ver en la pantalla.

HTML, o Lenguaje de Marcado de Hipertexto (HyperText Markup Language), es la base fundamental de la creación de páginas web. Se trata de un lenguaje de marcado que define la estructura y el contenido de una página, utilizando etiquetas para indicar cómo se debe mostrar cada elemento.

CSS, o Cascading Style Sheets (Hojas de Estilo en Cascada), es un lenguaje que se utiliza para dar estilo a las páginas web. Es decir, define cómo se deben mostrar los elementos HTML en una página web, como el color, la tipografía, el tamaño, el diseño y la posición.

JavaScript (abreviado como JS) es un lenguaje de programación de alto nivel que se utiliza para agregar interactividad y comportamiento dinámico a las páginas web.
